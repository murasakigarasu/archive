# Проект по курсу "Цифровая архивация современности" (2025-2026 учебный год). 
## Домашняя работа №3. Коллекция цифровых ресурсов

Репозиторий содержит коллекцию веб-сайтов, архивированных с помощью инструмента wpull для загрузки файлов. Описание метаданных получены с помощью metawarс. 
Также проведен результат анализа коллекции с помощью ресурса ArchiveReady по метрикам CLEAR.

## Краткое описание коллекции

В коллекцию вошли сайты научных ассоциаций России

1. [Ассоциация японоведов](http://japanstudies.ru/)
2. [Ассоциация медицинских антропологов](https://amarussia.info/)
3. [Ассоциация в области исследователей исторической информатика "История и компьютер"](https://aikhisc.ru/)
4. [Ассоциация цифровых гуманитарных наук](https://dh.sfu-kras.ru/)
5. [Альянс цифровых гуманитарных проектов](https://alliance4humanities.digital/)

## Методы сбора и архивации

В данной работе мы использовали несколько инструментов для создания полноценной коллекции веб-архивов.

1. Для сохранения архива мы воспользовались wpull. Этот инструмент представляет из себя утилиту командной строки а также библиотеку Python и позволяет загружать архивы с широким выбором заданных параметром (глубина рекурсии, timeout, метаданные архивирующего лица итд). В данной работе для загрузки нескольких архивов последовательно, мы воспользовались wpull через Python-скрипт массовой загрузки

2. Воспроизвести полученные архивы удалось с помощью сервиса ReplayWeb.page,

3. Для каждого архива мы проанаизировали качество полученных материалов с помощью инструмента ArchiveReady,

4. Наконец, для каждого архива мы произвели анализ метаданных с помощью Python-библиотеки metawarc

## Структура репозитория

1. Отдельные папки по каждому сайту
  -README.md: Ход работы по архивированию с wpull, анализа CLEAR метрик в ArchiveReady, анализ метаданных в metawarc
  -[название сайта]_meta.jsonl: Файл с подсчетом метаданных.
  -...
