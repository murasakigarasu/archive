# Архив сайта alliance4humanities.digital
## Об ассоциации
Альянс цифровых гуманитарных проектов был создан недавно, в декабре 2024 года с целью поддержки различных начинаний в сфере Digital Humanities. Архивация данного сайта представляет интерес для дальнйшего сравнения: насколько изменится наполненение и основные цели организации спустя время; будут ли они совпадать или сильно изменятся?

## Архивация
Архивация проходила при использовании утилиты wpull. Успешность оценивалась при помощи сервиса REPLAY WEBPAGE. Некоторые разделись могут отображаться некорректно, но основная информация (краткие биографические справки об участниках, новости о важных событиях, работы ведущих специалистов) восстановима.

## Показатели архивируемости
Сайт получил низкий индекс архивируемости – 56% по методике CLEAR с помощью сервиса ArchiveReady. Cohesion составляет 0%. Во-многом, как мне кажется, это объясняется динамической структурой сайта. Если посмотреть на скриншот выше, то видно, что при воспроизведении на сервисе REPLAY WEBPAGE сайт потреял все красивости, но при этом, однако, содержание осталость целым.

## metawarc
При работе с питоновской библиотекой metawarc использовались следующие команды:

1.
   ```{python, eval = FALSE}
   metawarc analyze
   ```

2.
   ```{python, eval = FALSE}
    metawarc metadata --output #[указание директории и название получаемого файла] [указание директории и название файла для форматирования]
    ```

4.
   ```{python, eval = FALSE}
   metawarc index [указание директории и название файла]
   ```

4.
   ```{python, eval = FALSE}
   metawarc stats -m exts
   ```
