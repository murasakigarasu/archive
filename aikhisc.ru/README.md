# Архив сайта aikhisc.ru
## Об ассоциации
Ассоциация "История и компьютер" - одна из старейших научных ассоциаций России с собственным сайтом, на котором хранятся архивы с начала 1990-х годов. Однако, в последнее время пополняется не так активно: у многих преподавателей кафедры исторической информатики свои сторонние проекты, плавный переход в телеграм и другие социальные сети. Поэтому, я бы внес его в список "находящиеся под угрозой закрытия/сильной модернизации" (особенно учитывая академическую ккультуру).

## Архивация
Архивация проходила при использовании утилиты wpull. Успешность оценивалась при помощи сервиса REPLAY WEBPAGE. Некоторые разделись могут отображаться некорректно, но основная информация (краткие биографические справки об участниках, новости о важных событиях, работы ведущих специалистов) восстановима.

## metawarc
При работе с питоновской библиотекой metawarc использовались следующие команды:

1.
   ```{python, eval = FALSE}
   metawarc analyze
   ```

2.
   ```{python, eval = FALSE}
    metawarc metadata --output #[указание директории и название получаемого файла] [указание директории и название файла для форматирования]
    ```

4.
   ```{python, eval = FALSE}
   metawarc index [указание директории и название файла]
   ```

4.
   ```{python, eval = FALSE}
   metawarc stats -m exts
   ```
